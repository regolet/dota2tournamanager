Registration Help
How to use the Registration Manager:
  1. Click Create Tournament Registration to open registration
  2. Set the desired expiry time (default is 24 hours)
  3. Share the registration link with participants
  4. Monitor registration status in real-time
  5. Close registration manually at any time
Tip: You can extend registration by creating a new registration period
